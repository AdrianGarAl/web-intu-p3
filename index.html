Responsive design

Fundamentos aplicados en la interfaz

Estos bloques ejemplifican los principios desarrollados en la memoria: rejillas fluidas, recursos flexibles, media queries y enfoque mobile first.

01

Rejilla fluida

Las tarjetas usan una cuadrícula flexible que aumenta o reduce el número de columnas según el espacio disponible. En móvil se apilan para evitar desplazamiento horizontal.

grid-template-columns: repeat(auto-fit, minmax(...))
02

Recursos flexibles

Los elementos gráficos se ajustan al contenedor y se evita depender de anchuras fijas. La composición mantiene proporciones estables en diferentes tamaños.

max-width: 100%
03

Media queries

Los puntos de ruptura modifican el menú, la distribución de columnas y el espaciado cuando el contenido empieza a necesitar otra presentación.

@media (max-width: 760px)
04

Mobile first

La base del diseño prioriza una experiencia simple y funcional. Desde ahí se añaden mejoras visuales en pantallas más grandes.

min-width / progressive enhancement

Usabilidad

Componentes diseñados para ser utilizables

La adaptación se evalúa por la facilidad para realizar tareas. Estos elementos muestran decisiones concretas sobre navegación, legibilidad, interacción táctil, formularios y accesibilidad.

Controles táctiles

Los botones tienen altura suficiente y separación visible para reducir pulsaciones erróneas.

Formulario breve

El formulario usa etiquetas visibles, tipos de campo adecuados y una estructura de una columna en móvil.

Jerarquía visual

El contenido se organiza por títulos, subtítulos, tarjetas y llamadas a la acción para facilitar el escaneo.

Comprobaciones de adaptación

La interfaz puede comprobarse en distintos tamaños de pantalla para observar cómo cambian la navegación, las columnas, la prioridad del contenido y el comportamiento de los controles.

Escritorio

Comprobar navegación horizontal, rejilla con varias columnas, panel adaptativo completo y uso del espacio.

Tableta

Comprobar transición intermedia, columnas reducidas, legibilidad y conservación de contenido principal.

Móvil

Comprobar menú desplegable, una sola columna, controles táctiles y ausencia de desplazamiento horizontal.
